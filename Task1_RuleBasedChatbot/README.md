# Rule-Based Chatbot using Python

A simple AI-inspired chatbot that responds to user queries using predefined rules and pattern matching techniques.

## Features

- Greeting responses
- Name identification
- Date and time information
- Help command
- Exit functionality
- Unknown query handling
- Easy to extend with additional rules

## Technologies Used

- Python 3
- Regular Expressions (re)
- Datetime Module

## Project Structure

```
rule-based-chatbot-python/

```

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/rule-based-chatbot-python.git
```

2. Navigate to the project folder

```bash
cd rule-based-chatbot-python
```

3. Run the chatbot

```bash
python chatbot.py
```

## Sample Interaction

```
You: Hello
Bot: Hello! How can I help you?

You: What is your name?
Bot: My name is SmartBot.

You: Time
Bot: Current time is 10:30:15

You: Bye
Bot: Goodbye!
```
